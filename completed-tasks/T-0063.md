# T-0063: Реализовать `SpriteFrames` и `AnimatedSprite2D`

- Статус: завершена
- Завершена: 2026-06-21T11:58:00+03:00
- Приоритет: P0
- Зависимости: T-0026, T-0035
- Домен: Анимация

## Исходная цель

Реализовать runtime baseline для `SpriteFrames` и `AnimatedSprite2D`, чтобы `0.1.0 Preview` поддерживал frame timing, loop, autoplay и реакцию на изменение `SpriteFrames` resource.

## Что сделано

- Добавлен публичный `SpriteFrames : Resource`.
- Добавлен публичный `SpriteFrames.LoopModeEnum` со значениями `None`, `Linear`, `Pingpong`.
- `SpriteFrames` поддерживает named animations, stable sorted names, add/remove/rename/duplicate, clear/clear all, frame insert/replace/remove, frame duration, animation speed и loop helpers.
- Добавлен публичный `AnimatedSprite2D : Node2D`.
- `AnimatedSprite2D` поддерживает `SpriteFrames`, `Animation`, `Autoplay`, `Frame`, `FrameProgress`, `SpeedScale`, `Centered`, `Offset`, `FlipH`, `FlipV`.
- Реализованы `Play()`, `PlayBackwards()`, `Pause()`, `Stop()`, `IsPlaying()`, `GetPlayingSpeed()`, `SetFrameAndProgress()`, `GetRect()` и `IsPixelOpaque()`.
- `_Ready()` запускает `Autoplay`, а `_Process(delta)` продвигает кадры перед draw traversal.
- `CanvasSubmissionContext` отправляет текущий frame texture для `AnimatedSprite2D`.
- Замена texture через `SpriteFrames.SetFrame()` видна в следующем canvas submission без пересоздания node.
- `Sprite2D` XML summary обновлён: frame-based animation теперь ведёт через `AnimatedSprite2D` и `SpriteFrames`.
- Добавлены specification, implementation documentation, Wiki/API compatibility rows и public type guard updates.

## Важные решения

- `AnimatedSprite2D` наследуется от `Node2D`, а не от `Sprite2D`, чтобы public API не получал прямые свойства `Texture`/`Region*`.
- `GetPlayingSpeed()` возвращает `SpeedScale * customSpeed` с учётом текущего pingpong direction; animation FPS учитывается отдельно при расчёте продвижения frame.
- `SpriteFrames.GetAnimationNames()` возвращает `StringName[]`, потому что packed string arrays ещё не входят в текущий public baseline.
