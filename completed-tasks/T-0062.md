# T-0062: Добавить debug visualization hooks для collision shapes и physics diagnostics

- Статус: завершена
- Завершена: 2026-06-21T11:35:00+03:00
- Приоритет: P1
- Зависимости: T-0055, T-0080
- Домен: Физика 2D

## Исходная цель

Добавить runtime hook для будущего editor viewport, чтобы collision shapes можно было включать для debug visualization и diagnostics checks без раскрытия внутренних объектов физической реализации в public API.

## Что сделано

- Добавлено публичное свойство `SceneTree.DebugCollisionsHint`.
- Добавлено публичное свойство `CollisionShape2D.DebugColor`.
- Добавлен internal `PhysicsDebugShape2D` snapshot: владелец collision object, shape node, shape resource, shape index, AABB bounds в мировых координатах, debug color и disabled flag.
- Snapshot возвращает пустой список, если `DebugCollisionsHint == false`.
- Snapshot включает disabled shapes, пропускает `CollisionShape2D` без `Shape`, заменяет `Color.Transparent` на default debug color и не раскрывает native pointers или изменяемые backend objects.
- Integration tests добавлены до production-кода; red-state run ожидаемо упал на отсутствующих API/типах, green-state run прошёл после реализации.
- Обновлены specification, implementation documentation, release baseline и GitHub Wiki source compatibility table.
