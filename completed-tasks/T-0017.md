## Итог

Реализован Godot-like 2D math baseline:

- `Vector2`;
- `Vector2I`;
- `Rect2`;
- `Rect2I`;
- `Transform2D`;
- `Color`;
- `Mathf`.

Публичный namespace остался `Electron2D`. Физически runtime-файлы разнесены по доменным папкам:

- `src/Electron2D/Core/Math/` - math value types и helpers;
- `src/Electron2D/Core/ObjectModel/` - `Object`, `RefCounted`, `Resource`, `Callable`, signal/error baseline;
- `src/Electron2D/Core/SceneTree/` - `Node`, `SceneTree`, `NodePath`, `PackedScene`, traversal/deferred/diagnostics internals.

## Проверенное поведение

- `Vector2` покрывает арифметику, длину, нормализацию, dot/cross, расстояние, direction, rotation, interpolation, clamp и invariant formatting.
- `Vector2I` покрывает integer arithmetic, remainder, length/aspect, component helpers и conversions.
- `Rect2` и `Rect2I` покрывают `End`, area/center, `Abs()`, point containment, intersections, merge и grow.
- `Transform2D` покрывает identity, translation, rotation, scale, inverse и transform composition.
- `Color` покрывает RGBA value semantics, interpolation, light/dark helpers и HTML parsing/formatting.
- `Mathf` покрывает constants, approximate equality, clamp, interpolation, angle conversion, positive modulo, snapping и integer rounding helpers.
- Public API compatibility table обновлена; runtime assembly теперь экспортирует `18` публичных типов.
