## Итог

Введена начальная Godot-like граница `PhysicsServer2D` для `0.1.0 Preview`. Публичный API работает через `Rid` и Godot-like enum-значения, а backend остаётся внутренним и заменяемым для runtime и тестов.

## Проверенное поведение

- `PhysicsServer2D.SpaceCreate()` создаёт space RID.
- `SpaceSetActive()`/`SpaceIsActive()` сохраняют active state space.
- `SpaceSetParam()`/`SpaceGetParam()` выполняют round-trip `SpaceParameter`.
- `AreaCreate()`, `BodyCreate()` и `JointCreate()` создают opaque RID для будущих операций.
- Shape factory methods создают shape RID, а `ShapeGetType()` возвращает ожидаемый `ShapeType`.
- `FreeRid()` освобождает RID; дальнейшее обращение к освобождённому RID завершается `ArgumentException`.
- Internal `IPhysicsServer2DBackend` можно заменить в тесте без public API для пользователя.
- Reflection-тест подтверждает отсутствие публичных `Box2D` типов и public signatures с backend handles.
