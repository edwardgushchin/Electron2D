## Итог

Реализован baseline path resolution для целевого API:

- добавлен public type `Electron2D.NodePath`;
- добавлены `Node.GetNode(NodePath path)` и `Node.GetNodeOrNull(NodePath path)`;
- `NodePath` парсит node names, subnames, absolute flag и empty state;
- C# shorthand `GetNode("Child")` работает через implicit conversion из `string` в `NodePath`;
- `SceneTree.Root` теперь имеет имя `root`, чтобы absolute paths вида `/root/...` совпадали с целевым соглашением.

## Проверенное поведение

- `NodePath` парсит `/root/World/Player:position:x` как absolute path с names и subnames.
- Relative lookup поддерживает direct child, nested child, `.`, `..` и sibling path через parent.
- Absolute lookup поддерживает `/root` и descendant paths от `SceneTree.Root`.
- Subnames после `:` игнорируются node lookup на текущем baseline.
- Missing path возвращает `null` через `GetNodeOrNull()` и `InvalidOperationException` через `GetNode()`.
- После rename node старый path не разрешается, новый path разрешается.
