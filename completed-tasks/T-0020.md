# T-0020 [x] P0: Реализовать закрытый список `Variant` для `0.1.0`

- Создана: 2026-06-20T16:16:20+03:00
- Завершена: 2026-06-21T01:37:00+03:00
- Статус: завершена
- Коммит: `T-0020: реализовать Variant baseline`

## Итог

Реализован `Electron2D.Variant` из целевого API как closed-list value carrier для `0.1.0 Preview`. Добавлены публичные коллекции `Electron2D.Collections.Array` и `Electron2D.Collections.Dictionary`, чтобы массивы и словари не подменялись произвольными CLR-контейнерами.

## Проверенное поведение

- `default(Variant)` и `Variant.CreateFrom(null)` дают `Variant.Type.Nil`.
- `bool`, integer-типы, `float`/`double`, `string` и enum values корректно приводятся к целевому `Variant.Type`.
- 2D math-типы `Vector2`, `Vector2I`, `Rect2`, `Rect2I`, `Transform2D`, `Color` хранятся и читаются строгими `As...()` методами.
- Identity и dynamic API значения `StringName`, `NodePath`, `Rid` и `Callable` покрыты.
- `Object`, `Resource` и `Node` хранятся как `Variant.Type.Object`.
- `Electron2D.Collections.Array` хранит ordered `Variant` values.
- `Electron2D.Collections.Dictionary` хранит пары `Variant` -> `Variant`.
- Unsupported CLR values дают `ArgumentException`, переполнение `ulong` даёт `ArgumentOutOfRangeException`, неверный `As...()` cast даёт `InvalidCastException`.
- Public API compatibility baseline обновлён до `25` публичных типов.

## Заметки агента

`Signal`, 3D-типы, packed arrays и stable serialization round-trip не включались в `T-0020`. Stable serialization остаётся задачей `T-0021`, а перевод signal/group/deferred/property APIs на `Variant` должен идти отдельными migration-задачами.
