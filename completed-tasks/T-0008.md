# T-0008: Реализовать базовые типы `Object`, `RefCounted`, `Resource`

- Статус: завершена
- Завершена: 2026-06-20T22:00:00+03:00
- Коммит: `T-0008: реализовать базовые runtime-типы`

## Итог

- Добавлены public Godot-like baseline-типы:
  - `Electron2D.Object`
  - `Electron2D.RefCounted`
  - `Electron2D.Resource`
- `Object` поддерживает `GetInstanceId()`, `Free()` и `Object.IsInstanceValid()`.
- `RefCounted` поддерживает `Reference()`, `Unreference()` и `GetReferenceCount()`.
- `Resource` поддерживает `ResourceName`, `ResourcePath`, `ResourceLocalToScene`, `ResourceSceneUniqueId`, `TakeOverPath()`.
- Добавлена спецификация `docs/specifications/object-model/base-object-lifetime.md`.
- Добавлена документация `docs/documentation/object-model/base-object-lifetime.md`.
- GitHub Wiki compatibility table обновлена: public type count теперь `3`.
- Changelog, release notes и README обновлены под новый public API baseline.

## Проверки

- До реализации `dotnet test tests\Electron2D.Tests.Unit\Electron2D.Tests.Unit.csproj --no-restore` упал ожидаемо: `Electron2D.Object`, `RefCounted`, `Resource` отсутствовали.
- После реализации `powershell -ExecutionPolicy Bypass -File tools/Run-Tests.ps1` прошёл успешно.
- `powershell -ExecutionPolicy Bypass -File tools/Verify-ApiCompatibility.ps1` прошёл успешно: `Public types: 3`.
- `powershell -ExecutionPolicy Bypass -File tools/Verify-ReleaseMetadata.ps1` прошёл успешно.
- `powershell -ExecutionPolicy Bypass -File tools/Verify-ProjectTemplate.ps1` прошёл успешно.
- `powershell -ExecutionPolicy Bypass -File tools/Verify-CiMatrix.ps1` прошёл успешно.
- `powershell -ExecutionPolicy Bypass -File tools/Verify-PerformanceBudgets.ps1` прошёл успешно.

## Отличия от Godot

- Metadata API отложен до появления `StringName` и `Variant`.
- `Resource.GetRid()` отложен до появления `Rid`.
- `IDisposable` не добавлен в public API.
