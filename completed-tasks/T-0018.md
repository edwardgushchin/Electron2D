## Итог

Реализован `Electron2D.RandomNumberGenerator` как Godot-like `RefCounted`-тип для воспроизводимых псевдослучайных последовательностей. Публичный API включает `Seed`, `State`, `Randomize()`, `Randi()`, `RandiRange()`, `Randf()`, `RandfRange()` и `Randfn()`.

## Проверенное поведение

- `Seed` переинициализирует PCG32 sequence policy.
- Для `Seed = 42` зафиксирована документированная последовательность `Randi()`: `3270867926`, `1795671209`, `1924641435`, `1143034755`, `4121910957`.
- Сохранённый `State` восстанавливает continuation последовательности.
- `RandiRange()` и `RandfRange()` поддерживают inclusive bounds, одинаковые границы и reversed bounds.
- `Randfn()` детерминирован для одного seed и возвращает finite значение; `deviation = 0` возвращает `mean`.
- Public API compatibility baseline обновлён до `19` публичных типов.
