## Итог

Зафиксирована базовая модель C# scripting для `0.1.0 Preview`: script class является обычным C# классом, наследуется от `Node`, компилируется обычной .NET toolchain вместе с проектом и получает Godot-like lifecycle callbacks без runtime compilation, dynamic assembly load или iOS dynamic code path.

## Проверенное поведение

- Script class наследуется от `Node`.
- `_EnterTree()` и `_Ready()` вызываются при добавлении script node в `SceneTree.Root`.
- `_Process(double)` и `_PhysicsProcess(double)` вызываются через текущий traversal host.
- Script получает доступ к дереву через `GetTree()`.
- Script получает доступ к runtime services через public facade `RenderingServer`.
- Шаблон `templates/electron2d-empty` содержит `Scripts/MainScene.cs` и проверяемый script lifecycle sample.
- `tools/Verify-ProjectTemplate.ps1` восстанавливает свежий local package в изолированный packages folder и проверяет output script sample.
- Public API не расширялся: используется уже существующая Godot-like `Node`/`SceneTree`/`RenderingServer` модель.
