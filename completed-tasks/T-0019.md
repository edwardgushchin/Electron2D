## Итог

Реализованы базовые целевые identity-типы `Electron2D.StringName` и `Electron2D.Rid`, а также internal `RidAllocator` для будущих server-backed resources. В этом контексте `Rid` означает opaque resource identifier текущей runtime-сессии, а не .NET Runtime Identifier для платформ сборки.

## Проверенное поведение

- `StringName` поддерживает equality, hashing, `ToString()`, `IsEmpty()`, implicit conversion из `string?`, `default`, `null` и empty semantics.
- `StringName` использует ordinal equality и работает как key в `Dictionary<StringName, TValue>`.
- `Rid` поддерживает invalid/default ID `0`, `GetId()`, `IsValid()`, equality, ordering, hashing и `ToString()`.
- Internal `RidAllocator` выдаёт уникальные valid `Rid`, отслеживает ownership текущей runtime-сессии и освобождает ownership через `Free()`.
- Public API compatibility baseline обновлён до `21` публичного типа.
