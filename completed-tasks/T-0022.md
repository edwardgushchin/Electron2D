## Итог

Введён `RenderingServer` facade из целевого API с `CurrentProfile`, `HasFeature()` и nested renderer profile/feature enums. Concrete backend boundary оставлена internal: `IRenderingBackend`, `RenderingBackend`, `CompatibilityRenderingBackend`, `StandardRenderingBackend`.

## Проверенное поведение

- Default runtime profile - `Compatibility`.
- Compatibility profile включает базовые features: sprites, animation, tile map, UI, text, primitives, camera, clipping и standard blend modes.
- Compatibility profile не сообщает standard-only features: custom shaders, render targets и post-processing.
- Internal `StandardRenderingBackend` переключает `CurrentProfile` на `Standard` и включает standard-only features.
- Internal `CompatibilityRenderingBackend` переключает `CurrentProfile` обратно на `Compatibility` и отключает standard-only features.
- Public node surface не раскрывает concrete rendering backend.
- Internal backend interfaces/classes не экспортируются из public assembly.
- Public API compatibility baseline обновлён до `28` публичных типов.
