## Итог

Правило MIT License закреплено в локальных инструкциях агента. Следующие агенты должны создавать и редактировать source-файлы так, чтобы C# и PowerShell файлы сохраняли полный MIT license header в стиле SDL: текст лицензии находится прямо в начале каждого файла.

## Проверенное поведение

- `AGENTS.md` прямо указывает, что Electron2D распространяется по MIT License.
- `AGENTS.md` запрещает подменять MIT текстом SDL zlib license: аналогия с SDL означает формат per-file header, а не другую лицензию.
- Для C# файлов требуется header в форме `/* ... */`.
- Для PowerShell файлов требуется header в форме `<# ... #>`.
- Перед staging или commit source-изменений требуется запуск `tools\Verify-SourceLicenseHeaders.ps1`.
