## Итог

Реализован Godot-like direct 2D physics query baseline для `0.1.0 Preview`: `World2D`, `PhysicsDirectSpaceState2D`, query parameter resources и исполняемый `RayCast2D` работают поверх managed AABB-проверок активных `CollisionShape2D`.

## Проверенное поведение

- `CanvasItem.GetWorld2D()` возвращает `World2D`.
- `World2D.DirectSpaceState` возвращает `PhysicsDirectSpaceState2D`.
- `PhysicsDirectSpaceState2D.IntersectRay()` возвращает ближайший hit dictionary или пустой dictionary для no-hit.
- `PhysicsDirectSpaceState2D.IntersectPoint()` и `IntersectShape()` возвращают `Array` of result dictionaries.
- Query parameters поддерживают masks, body/area flags, excluded RIDs и no-hit сценарии.
- `RayCast2D.ForceRaycastUpdate()` и physics frame обновляют cached collision result.
- Public API не раскрывает Box2D.NET или другой backend handle.
