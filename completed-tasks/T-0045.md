## Итог

Реализован script metadata baseline для `0.1.0 Preview`: публичные marker attributes целевого API `[Export]`, `[Signal]`, `[Tool]` и internal AOT-safe bridge для serialization/Inspector без runtime assembly scan, legacy compatibility layer или Unity-like component model.

## Проверенное поведение

- `ExportAttribute` доступен как `[Export]` для fields/properties.
- `SignalAttribute` доступен как `[Signal]` для delegate declarations.
- `ToolAttribute` доступен как `[Tool]` для script classes.
- Публичные атрибуты не добавляют Electron2D-specific public properties или methods.
- Exported properties round-trip через `ScriptObjectSerializer` и `SerializedPropertyValue`.
- Runtime-only properties, отсутствующие в metadata, не меняются при restore.
- Signals из metadata регистрируются на script instance и вызываются через `Connect()`/`EmitSignal()`/`Callable`.
- `[Tool]` metadata помечается как experimental и sandboxed.
- Runtime не сканирует assemblies и не делает automatic public property discovery.
