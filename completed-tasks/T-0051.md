## Итог

Реализован mobile input baseline для `0.1.0 Preview`: touch press/release/cancel, touch drag, mobile back/menu navigation, virtual keyboard request state, screen orientation state и display safe area state.

## Проверенное поведение

- Touch down/up/cancel доставляются в `_Input()` как `InputEventScreenTouch`.
- Touch motion доставляется в `_Input()` как `InputEventScreenDrag`.
- Invalid touch device/finger ids fail closed и не создают public events.
- Back/menu navigation доставляется как `InputEventKey` с `Key.Back` и `Key.Menu`.
- `DisplayServer.ScreenSetOrientation()` и platform orientation updates меняют process-wide orientation state.
- `DisplayServer.GetDisplaySafeArea()` возвращает последний platform safe area snapshot.
- `DisplayServer.VirtualKeyboardShow()`, `VirtualKeyboardHide()` и `VirtualKeyboardGetHeight()` хранят безопасное process-wide состояние без public native handles.
- Committed text input остаётся на existing path через `InputEventKey.Unicode`.
