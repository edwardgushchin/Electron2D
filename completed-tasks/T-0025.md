## Итог

Введён baseline публичных texture resources целевого API: `Texture2D` и `AtlasTexture`. Добавлен internal texture upload/reload/release registry с sampling descriptor и leak tracking, проверенный runtime smoke тестом.

## Проверенное поведение

- `Texture2D` экспортирует целевой metadata API: `GetWidth()`, `GetHeight()`, `GetSize()`, `HasAlpha()`, `HasMipmaps()`, `GetMipmapCount()`, `IsPixelOpaque()`.
- `AtlasTexture` хранит `Atlas`, `Region`, `Margin` и `FilterClip`.
- `AtlasTexture` использует region size, а нулевая axis size fallback-ится к atlas size.
- `AtlasTexture.IsPixelOpaque()` переводит координаты через `Region.Position` и делегирует atlas texture.
- Internal `TextureResourceRegistry` выполняет upload, reload и release через `ITextureGpuApi`.
- Internal `TextureUploadDescriptor` содержит size, transparency, mipmaps, source region, margin, filter clip и sampling options.
- Ошибка upload не оставляет active texture handle и пишет `TextureResourceEventKind.Error`.
- Runtime smoke test проверяет upload -> reload -> release cycle и `LeakCount == 0`.
- Public API compatibility baseline обновлён до `30` публичных типов.
