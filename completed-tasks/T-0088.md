## Самодостаточное описание

Реализован первый Windows x64 export baseline для `0.1.0 Preview`: внутренний planner строит deterministic package plan для `win-x64`, Debug/Release, self-contained publish, renderer profile и window/fullscreen state. Отдельный Windows-only verifier публикует шаблонный проект и запускает exported executable, чтобы проверить reference scene и C# script lifecycle.

Реализация не добавляет public runtime API, не подписывает artifacts, не выполняет deploy и не публикует GitHub Release.

## Решения

- Windows export code размещён в `src/Electron2D/Export/Windows`, а не в `Core`.
- Planner не запускает внешние процессы: он строит проверяемый plan, а реальный publish/run выполняет отдельный verifier.
- CI запускает Windows verifier только на Windows runner, чтобы Linux/macOS jobs не пытались публиковать `win-x64`.
