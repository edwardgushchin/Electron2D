## T-0046 [ ] P0: Реализовать создание скрипта, attach к узлу, встроенный редактор кода и сборку проекта из редактора

- Создана: 2026-06-20T16:16:20+03:00
- Приоритет: P0
- Зависимости: T-0044, T-0081
- Ссылки:
  - Upstream commit: нет
  - Спецификация: `docs/specifications/releases/0.1.0-preview.md`; `docs/specifications/scripting/`
  - Документация: `docs/documentation/scripting/`
  - Исходный код: `src/Electron2D/`; `src/Electron2D.Editor/`; `src/Electron2D/Runtime/`; `examples/`

### Самодостаточное описание

Задача относится к домену «C# scripting». Цель - реализовать создание скрипта, attach к узлу, встроенный редактор кода для написания C#-скриптов и сборку проекта из редактора. Текущее состояние: runtime написан на C#, но полноценная модель пользовательских C#-скриптов, export metadata и связь с редактором ещё не выделены как релизный путь. Нужное поведение: результат должен закрыть релизный контракт `0.1.0 Preview`, быть проверяемым автоматическими тестами или документированной проверкой и соответствовать критериям: встроенный редактор кода сохраняет скрипт, editor shows compiler errors and can rerun scene after rebuild.

Важное решение из документации: первая версия поддерживает только C#; runtime-компиляция и динамическая загрузка пользовательского кода на iOS не должны становиться обязательным условием. Ограничение для агента: не вводить GDScript, visual scripting, обязательный Hot Reload, внешний IDE workflow как обязательное требование или полноценную C# IDE внутри редактора в 0.1.0. Встроенный редактор кода должен быть достаточен для создания, редактирования, сохранения и сборки скриптов. Не менять несвязанные файлы, не расширять scope за пределы `0.1.0 Preview`, не закрывать задачу без проверяемого результата и не подменять обязательные тесты или документацию устным утверждением.
