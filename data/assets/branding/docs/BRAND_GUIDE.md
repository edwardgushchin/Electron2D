# Electron2D Brand Guide

## Primary tagline

**AGENT-NATIVE CROSS-PLATFORM 2D GAME ENGINE**

## Palette

- Ink: `#101625`
- Navy: `#0E1422`
- Violet: `#6C5CFF`
- Purple: `#B44DFF`
- Pink: `#FF58C7`
- Muted light: `#667085`
- Muted dark: `#AEB7C8`

## Usage

- `logo/` — horizontal primary logo for general use.
- `readme/` — GitHub README versions for light and dark themes.
- `icon/` — Windows `.ico`, source SVG and PNG sizes from 16 to 1024 px.
- `social/` — cropped transparent logo variants in dark-text and light-text variants.
- `mark/` — standalone infinity/electron symbol.
- Files ending in `_editable.svg` retain editable text.
- SVG files without `_editable` have text converted to vector paths.

## Repository normalization

PNG and SVG logo variants in `logo/`, `readme/`, `social/` and the transparent `mark/` are cropped to the visible logo bounds. Fixed-size Windows icon assets in `icon/` and the contact sheet in `preview/` keep their original canvas.

## Clear space

Keep free space around the logo at least equal to the diameter of the electron dot.

## Vertical alignment

The heading and subtitle form one compact block. Their visible combined height matches the standalone symbol.

## Width alignment

The subtitle is aligned to the same left edge as `Electron2D` and is tuned to the same visible width.

## Do not

- Separate `2D` from `Electron`.
- Stretch the symbol or wordmark independently.
- Add a background inside the transparent logo variants.
- Recolor only one half of the infinity mark.
