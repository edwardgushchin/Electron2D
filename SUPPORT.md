# Support

Use the channel that matches the kind of help you need:

- Documentation and guides: [Electron2D GitHub Wiki](https://github.com/edwardgushchin/Electron2D/wiki)
- Bugs: [GitHub Issues](https://github.com/edwardgushchin/Electron2D/issues/new/choose)
- Feature requests and design feedback: [GitHub Issues](https://github.com/edwardgushchin/Electron2D/issues/new/choose)
- Broader questions and discussions: [GitHub Discussions](https://github.com/edwardgushchin/Electron2D/discussions)

Do not share secrets, private keys, tokens, private customer data or production credentials in public support requests.
